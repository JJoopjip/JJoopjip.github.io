Project 01

Job Application Tracker

A personal command centre for a serious job hunt. Every application lives on one board — with a daily dashboard that tells me what actually needs attention, so nothing slips.

Built with Python · Flask Runs locally on my machine Feeds from the Résumé Generator Source on GitHub ↗
Live demo

The real tracker runs on my own machine, so I rebuilt its interface here — it works right in this page. Drag a card between columns — or tap one on a phone — to see a status change save. Every application shown is an anonymous placeholder; my real ones stay private.

Kanban board

Every stage, one glance

Cards move through Wishlist → Applied → Screening → Interview → Final. Drag to change status on desktop, tap to update on a phone — it saves on drop.

Daily dashboard

Only what needs me

A “needs attention” feed surfaces anything overdue or gone quiet, so a busy week never means a dropped follow-up.

Inbox & backup

Everything in one place

Application details, links, and notes stay together, with a backup option so the record is never lost.

One-click intake

No re-typing

New applications arrive straight from the Résumé Generator — company, role, and documents already attached.

How it connects

The tracker doesn't store its own copy of a résumé or posting — it reads from the Résumé Generator, so a single source of truth drives both apps and nothing is duplicated. See the Résumé Generator →

Project 02

AI Résumé Generator

Paste a job posting and get a one-page résumé built from my real career history — plus, if I want, a matching cover letter and a coverage score that shows exactly how well I fit the role.

Built with Python · headless AI agent Runs locally on my machine Outputs PDF + Word Source on GitHub ↗
Live demo

The real generator runs locally and calls an AI model to write from my private career history, so I rebuilt its interface here. Press Generate to watch a real run replay — the log lines are the ones the actual pipeline prints. Nothing you type is sent anywhere.

Flexible output

Résumé, cover letter, or both

Choose résumé only, résumé and cover letter in one run, or add the cover letter afterward once the résumé looks right.

Coverage score

Know your fit before you apply

Each run analyses how well the résumé matches the posting, so I can see where the gaps are and decide whether — and how — to close them. Gaps get closed by adding real experience, never by inventing it.

Live step log

Nothing hidden

A running log shows each step as it reads the spec, my history, and the posting, then writes and renders the documents.

One-click hand-off

Straight to the tracker

Send the finished résumé, cover letter, and posting to the Application Tracker in a single click — no copy-paste.

How it connects

This is the source of truth for both tools. When I add an application to the tracker, it pulls from here — so the résumé, cover letter, and posting live in one place and never get duplicated. See the Application Tracker →
